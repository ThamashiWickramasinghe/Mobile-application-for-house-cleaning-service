Clean App is a cross-platform mobile application developed using **Flutter (Dart)** and **Firebase** to connect customers with professional cleaners. The app allows customers to post house cleaning jobs and manage vacancies, while cleaners can browse available jobs, apply, and manage their bookings.

Firebase is used for authentication, real-time data management, cloud storage, and notifications, ensuring scalability and efficient performance. The application supports role-based functionality for both customers and cleaners, providing a user-friendly and reliable cleaning service platform.
